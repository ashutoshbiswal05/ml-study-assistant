# ML Study Assistant (Capstone Project)

This project is a Retrieval-Augmented Generation (RAG) based AI agent that helps users understand machine learning concepts.

## Features
- LangGraph-based agent workflow
- ChromaDB vector database for retrieval
- Groq LLM for response generation
- Streamlit UI for interaction

## How to run
1. Install dependencies
2. Add your GROQ_API_KEY in a .env file
3. Run:
   streamlit run capstone_streamlit.py

## Topics Covered
- Overfitting, Bias-Variance, Regression, Clustering, Neural Networks, etc.
